DEPLOYMENT CHECKLIST
================================================================================

Pre-Deployment
--------------
□ Read README.md
□ Review QUICK-REFERENCE.md
□ Review relevant docs/*.md guides
□ Run ./scripts/validate.sh
□ Backup state: ./scripts/state-management.sh backup ENV

Deployment
----------
□ Review terraform plan output
□ Confirm environment-specific settings
□ Apply terraform: tofu apply -var-file=../environments/ENV.tfvars
□ Deploy applications via Ansible
□ Run health checks
□ Monitor logs via CloudWatch

Post-Deployment
---------------
□ Verify all services healthy
□ Check monitoring alerts active
□ Document any manual changes
□ Commit changes to Git
□ Update runbook if needed

================================================================================
SCRIPT QUICK START
================================================================================

Validate Configuration
  $ ./scripts/validate.sh
  Checks: Terraform, Ansible, Docker, secrets, permissions

Backup State
  $ ./scripts/state-management.sh backup prod

Rollback Infrastructure
  $ ./scripts/rollback.sh -e prod -t terraform --create-backup

Rollback Applications
  $ ./scripts/rollback.sh -e prod -t ansible --create-backup

View State
  $ ./scripts/state-management.sh view prod

Test LocalStack
  $ ./scripts/localstack.sh start
  $ ./scripts/localstack.sh deploy
